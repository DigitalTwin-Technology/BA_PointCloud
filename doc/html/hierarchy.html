 ▼CBAPointCloudRenderer.CloudController.AbstractPointCloudSetA PointCloudSet enables loading and rendering several point clouds at once.  ▼CBAPointCloudRenderer.Controllers.AbstractPointSetControllerA PointSetController enables loading and rendering several point clouds at once using an AbstractRenderer.  CBAPointCloudRenderer.ObjectCreation.DefaultMeshConfigurationThis is the default Mesh Configuration, that is able to render points as pixels, quads, circles and also provides fragment and cone interpolations using the fragment shader (see Thesis chapter 3.3.4 "Interpolation").
