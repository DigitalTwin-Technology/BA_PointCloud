Detailed Description

This struct can be used to describe the rendering priority. (See Bachelor Thesis chapter 3.1.5 "Loading Priority") The octree-level as well as the calculated priority value are considered. So (grand-)parents will have a higher priority than their (grand-)children. For nodes without such a relationship, the given priority-double-value determines the order.
